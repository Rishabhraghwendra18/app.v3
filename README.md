## Getting Started

First, setup the environment variables:
1. Create a file called .env.development at the root
2. Create 3 variables MORALIS_APPLICATION_ID, MORALIS_SERVER_ID, NETWORK_CHAIN

Run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
